## Featured Projects

### MealMate: Food Waste Reduction Platform
**MSc Dissertation Project (December 2025)**

An integrated web platform that combines machine learning based waste prediction with real time food redistribution to help restaurants reduce waste and connect surplus food with people who need it.

**Key Features:**
- Machine learning based waste prediction achieving an R² score of 0.75 on real world bakery data
- Interactive geospatial search with Google Maps integration
- Real time messaging system using WebSocket for pickup coordination
- Secure payment processing using Stripe escrow
- Analytics dashboard tracking waste trends and environmental impact

**Tech Stack:** React, Node.js, MongoDB, Python Flask, Socket.io, Stripe API, Google Maps API
